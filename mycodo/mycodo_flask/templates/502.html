Error 502: Bad Gateway Error

Something bad happened but it's probably not your fault. Try refreshing the page to see if it will load. If that doesn't work, try rebooting your system, then try refreshing it again. If that still doesn't work, submit a new issue on GitHub with any useful information that can help determine the cause of this issue.

The last 50 lines from the Mycodo frontend web log can be viewed with the command: 

journalctl -u mycodoflask | tail -n 50

If you create an issue on GitHub, provide this excerpt of the log for diagnostic purposes.
